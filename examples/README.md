# Examples

These examples show both the planning stage and the final generated Codex pet package.

## Available Examples

- [blue-robot-cat](blue-robot-cat): a small cyber-cute robot cat with screen-face expressions and clear action notes.
- [vegeta](vegeta): a real generated half-body Saiyan warrior pet with static frames and row GIFs.
- [dark-judge](dark-judge): a real generated head-only Saiyan-style pet with static frames and row GIFs.
- [wukong-spark](wukong-spark): a real generated Codex desktop pet package with `pet.json`, `spritesheet.webp`, full static frames, and row GIFs.

## How To Use An Example

Use an example when you want to explain the workflow to a beginner:

1. Open the preview card.
2. Read the brief.
3. Compare the 9 action notes with the official action names.
4. Start Codex and type `/create-pet` to make a new pet with the same structure.
